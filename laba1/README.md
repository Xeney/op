# Лабораторная работа №1

## Тема
Простейшие программы (переменные и константы, типы данных, математические операции, ввод и вывод данных в консоль)

## Цель
Научиться работать с переменными и константами различных типов данных, выполнять математические операции с использованием библиотеки Math, вводить и выводить данные в консоль с помощью класса Console.

## Основное задание
Создать консольное приложение на C#, которое решает математическое уравнение согласно варианту:

1. **Объявление переменных и констант**
   - Объявить константы для константных значений уравнения, числа π и числа e
   - Объявить переменные для значений a, b и промежуточных вычислений
   - Объявить переменные для хранения результатов

2. **Ввод данных**
   - Запросить у пользователя значения a и b через `Console.WriteLine()`
   - Использовать `Console.ReadLine()` и `Parse()` для преобразования ввода

3. **Математические вычисления**
   - Вычислить значение математической функции по варианту
   - Использовать методы класса Math

4. **Вывод результатов**
   - Вывести результаты через `Console.WriteLine()`
   - Отформатировать вывод с двумя знаками после запятой

## Варианты уравнений
1. \( f = \frac{\ln^2 b}{\cos\alpha - 1} \)
2. \( f = \pi \left( \frac{\ln b^5}{\sin\alpha + 1} \right) \)
3. \( f = \frac{\sin\alpha + tg\left( 2\alpha \right)}{\sqrt{\log_3 e^2}} \)
4. \( f = \frac{\cos^7 \pi + \sqrt{\ln\left(b^4\right)}}{\sin\left(\frac{\pi}{2} + \alpha\right)^2} \)
5. \( f = \sin^2 \left( \frac{\log_3 b}{\sqrt{\cos\left( 2\alpha \right)}} \right) \)
6. \( f = \sin\left( \frac{\alpha^3 + \beta^5}{2\pi} \right) + \sqrt{\cos\left(\alpha + \beta\right)} \)
7. \( f = \sin^2 \left( \frac{3\pi}{4} + \frac{\alpha}{3} \right) + \sqrt{\cos\beta} \)
8. \( f = -4\sin^3 \left( 3\alpha \right) + \frac{\sqrt{b}}{\ln\left(b+2\right)} \)
9. \( f = \sqrt{\frac{\sin^2 \alpha + \cos^3 \beta}{\sin^3 \alpha - \cos^2 \beta}} \)
10. \( f = \log_b \left( \frac{\sqrt{b + \sin \alpha}}{\cos^3 \alpha} \right) \)
