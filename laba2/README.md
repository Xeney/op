# Лабораторная работа №2

## Тема
Управляющие конструкции (условные операторы if-else, switch, циклы for, while, do while + контроль ввода с помощью метода TryParse)

## Цель
Научиться использовать управляющие конструкции языка C# для создания логики программ, включая условные операторы, циклы и контроль ввода, а также взаимодействие с пользователем через меню.

## Основное задание
Создать консольное приложение с меню выбора действий:

### 1. Меню выбора действий
- **Пункты меню:**
  1. Отгадай ответ
  2. Об авторе (вывод ФИО студента и группы)
  3. Выход

- **Реализация:**
  - Использование оператора `switch` для меню
  - Цикл `while` для создания цикличности
  - При выборе пункта 3 - запрос подтверждения выхода ("д"/"н")
  - Обработка некорректного ввода с повторным запросом

### 2. Контроль ввода
- Использование циклов `while` и метода `TryParse` для проверки корректности ввода
- Обработка исключительных ситуаций:
  - Деление на 0
  - Корень из отрицательного числа
  - Определение области допустимых значений для a и b

### 3. Игровая логика "Отгадай ответ"
- Перенос логики вычислений из лабораторной работы №1
- Ввод значений a и b с контролем корректности
- Ввод предполагаемого ответа пользователем с контролем корректности
- Сравнение ответа пользователя с вычисленным результатом (округление до 2 знаков)
- **3 попытки** для ввода правильного ответа
- Информирование о количестве оставшихся попыток
- При выигрыше/проигрыше - вывод соответствующего сообщения и возврат в меню

### 4. Ограничения
- Запрещено использование операторов `break`, `goto`, `continue` для выхода из циклов

## Дополнительное задание (необязательное)
**"Простой математический тест"**
- Добавление пункта в меню
- 3 случайных математических вопроса (сложение, вычитание, умножение)
- Генерация через класс `Random`
- Контроль ввода ответа с помощью `TryParse`
- Подсчет правильных ответов
- Вывод результата и возврат в главное меню
