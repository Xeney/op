# Лабораторная работа №3

## Тема
Одномерные массивы и методы

## Цель
Научиться работать с одномерными массивами, использовать методы для выполнения различных задач, включая сортировку массивов и сравнение времени выполнения различных алгоритмов сортировки.

## Основное задание
Создать консольное приложение на основе кода лабораторной работы №2 с добавлением функциональности:

### 1. Меню выбора действий
- Добавлен новый пункт меню "3. Сортировка массива"

### 2. Сортировка массива
- Запрос размера массива `n` у пользователя
- Создание массива из `n` элементов
- Заполнение массива случайными значениями с помощью класса `Random`
- Реализация двух методов сортировки согласно варианту
- Сравнение времени выполнения двух сортировок
- Вывод массива до и после сортировки
- **Ограничение**: при длине массива > 10 элементов выводить сообщение "Массивы не могут быть выведены на экран, так как длина массива больше 10"
- **Важно**: для сравнения сортировок необходимо клонировать исходный массив

### 3. Разбивка кода на методы
**Обязательные методы:**
- Метод для игры "Отгадай ответ"
- Метод для вывода информации об авторе
- Метод для сортировки массивов
- Метод для выхода (возвращает `bool` - закрыть программу или остаться)
- Метод для ввода и контроля ввода числа (универсальный, переиспользуемый)
- Метод для вычисления значения функции (принимает a, b, возвращает результат)
- Метод для логики игры "угадайка" (принимает правильный ответ)
- Метод для ввода длины массива (проверка, что длина > 0)
- Метод для создания и инициализации массива (принимает длину, возвращает массив)
- Метод для создания копии массива (принимает исходный, возвращает копию)
- Метод для вывода массива (проверка длины ≤ 10)
- Методы для сортировки массивов (принимают массив, возвращают отсортированный)

**Требования:**
- Все методы находятся за пределами метода `Main`
- Методы переиспользуемые, без повторений
