# Лабораторная работа №4

## Тема
Многомерные массивы

## Цель
Научиться работать с многомерными массивами в языке C#, применять их для решения сложных задач и создания игровых приложений с интерактивным взаимодействием через консоль.

## Основное задание
Создать консольное приложение на основе кода лабораторной работы №3 с добавлением игровой функциональности:

### 1. Меню выбора действий
- Добавлен новый пункт меню "4. Игра"

### 2. Требования к игре
- Реализация одной из предложенных игр с использованием многомерных массивов
- Корректное взаимодействие с пользователем
- Проверка условий победы/поражения

### 3. Игровое поле
- Представлено в виде двумерного или трёхмерного массива
- Обновление состояния после каждого хода игрока

### 4. Взаимодействие с пользователем
- Ввод данных через консоль
- Проверка корректности ввода данных

### 5. Логика игры
- Проверка условий окончания игры (победа, поражение, ничья)
- Вывод соответствующего сообщения по завершении игры

### 6. Разбиение программы на методы
**Обязательные методы:**
- Метод для инициализации игрового поля
- Метод для вывода текущего состояния поля
- Метод для ввода данных пользователем
- Метод для обработки хода и обновления состояния игры
- Метод для проверки условий окончания игры

### 7. Дополнительные требования (необязательные)
- Реализация нескольких уровней сложности

## Варианты игровых заданий

### 1. Морской бой
- **Поле**: 5×5
- **Корабли**: 1-, 2-, 3-палубные
- **Особенности**: ручное размещение игроком, случайное - компьютером, обработка попаданий/промахов

### 2. Тетрис
- **Поле**: 10×10
- **Фигуры**: различные формы блоков
- **Особенности**: управление движением, удаление заполненных рядов

### 3. Сапёр
- **Поле**: 5×5
- **Мины**: 5 штук
- **Особенности**: отображение количества мин вокруг клетки

### 4. Четыре в ряд
- **Поле**: 6×7
- **Цель**: собрать линию из 4 фишек
- **Особенности**: проверка по горизонтали, вертикали, диагоналям

### 5. Крестики-нолики на большом поле
- **Поле**: 10×10
- **Цель**: линия из 5 символов
- **Особенности**: проверка всех направлений

### 6. Змейка
- **Поле**: 20×20
- **Управление**: клавишами направления
- **Особенности**: рост при съедании еды

### 7. Шахматы
- **Поле**: 8×8
- **Фигуры**: пешки, ладьи, кони, король
- **Правила**: шахматные

### 8. Змейка с препятствиями
- **Поле**: 20×20
- **Особенности**: случайные неподвижные препятствия

### 9. Магический квадрат
- **Поле**: 3×3
- **Цель**: сумма строк/столбцов/диагоналей = 15
- **Числа**: 1-9 без повторений

### 10. Поиск сокровищ
- **Поле**: 10×10
- **Элементы**: сокровища и ловушки
- **Цель**: найти все сокровища, избегая ловушек

## Практические задачи

### Простые (1 обязательная вместо теории)
1. Сумма элементов в двумерном массиве
2. Максимальный элемент в трёхмерном массиве
3. Транспонирование матрицы
4. Сумма каждой строки в двумерном массиве
5. Сумма каждого столбца в двумерном массиве
6. Поиск значения в двумерном массиве
7. Преобразование двумерного массива в одномерный
8. Умножение двух матриц
9. Проверка симметрии двумерного массива
10. Диагональные элементы квадратной матрицы
11. Сортировка строк в двумерном массиве
12. Сумма диагональных элементов в двумерном массиве
13. Проверка на единичную матрицу
14. Реверс строк двумерного массива
15. Минимум в каждом столбце двумерного массива

### Сложные (1 обязательная)
1. Сумма элементов по главной и побочной диагоналям
2. Максимальный элемент в трёхмерном массиве с координатами
3. Нормализация матрицы
4. Сумма элементов каждой строки и столбца
5. Произведение двух матриц с проверкой совместимости
6. Нахождение локальных максимумов в двумерном массиве
7. Обратная матрица для квадратной матрицы
8. Поиск всех седловых точек в двумерном массиве
