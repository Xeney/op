# Лабораторная работа №5

## Тема
Работа с классами и XML-документированием

## Цель
Научиться структурировать код с помощью классов, обеспечить инкапсуляцию данных и разделение логики программ, а также применять XML-документирование для генерации описаний методов и классов.

## Основное задание
Создать консольное приложение на основе лабораторной работы №4 с добавлением классов и XML-документирования.

### Требования к классам:
1. **Статический класс для игры "Отгадай ответ"**
   - Метод для вычисления значения функции
   - XML-документирование всех публичных членов

2. **Класс для работы с массивами**
   - Поля: количество элементов, массив (private)
   - Конструктор без параметров (по умолчанию 10 элементов)
   - Конструктор с параметрами (пользовательский размер)
   - Методы для работы с массивами из лабораторной работы 3
   - Демонстрация создания экземпляров обоими конструкторами

3. **Статический класс для проверки вводимых данных**
   - Методы для контроля ввода

4. **Класс для реализации игры**
   - Константы: размер поля, обозначения элементов
   - Методы из лабораторной работы 4

### Принципы разработки:
- Принцип разделения логики работы и визуализации данных
- Поля классов с модификатором доступа `private`
- Использование свойств для доступа к полям из других классов
- XML-документирование всех классов и их публичных членов

## Теоретические вопросы (10)
1. Основные принципы ООП и их описание
2. Отличие класса от объекта
3. Модификаторы доступа в C# и их различия
4. Члены класса
5. Поля класса: синтаксис, уровень доступа
6. Конструктор класса: назначение, возвращаемые значения
7. Свойства: синтаксис, методы `get`/`set`
8. Отличие конструктора от метода
9. Создание экземпляра класса (на примере класса Circle)
10. Статические классы: отличия от нестатических, назначение

## Практические задачи (15)

### 1. Класс "Прямоугольник"
- Поля: длина, ширина (private)
- Свойства для доступа к полям
- Методы: площадь, периметр, проверка на квадрат
- Конструкторы: без параметров, с параметрами

### 2. Класс "Книга"
- Поля: название, автор, цена, год выпуска
- Свойства для доступа
- Метод для получения информации в формате строки
- Конструкторы: по умолчанию, со всеми полями, с основными данными

### 3. Класс "Студент"
- Поля: имя, возраст, список оценок
- Методы: добавление/удаление оценок, средний балл
- Конструкторы: только имя, имя и возраст, имя+возраст+оценки

### 4. Класс "Банк"
- Поля: номер счета, владелец, баланс
- Методы: пополнение, снятие, проверка баланса
- Конструкторы: по умолчанию, счет+владелец, все поля

### 5. Статический класс "Калькулятор"
- Методы: арифметические операции, корень, факториал

### 6. Классы "Точка" и "Линия"
- Point: конструкторы (без параметров, с координатами)
- Line: использование Point, метод расчета длины

### 7. Класс "Автомобиль"
- Поля: марка, модель, пробег, топливо
- Методы: заправка, поездка
- Конструкторы: по умолчанию, 2 параметра, 4 параметра

### 8. Классы "Товар" и "Инвентарь"
- Product: название, количество, цена
- Inventory: список продуктов, методы добавления/удаления/отображения
- Конструкторы Product: по умолчанию, название+цена, все поля

### 9. Классы "Круг" и "Цилиндр"
- Circle: радиус, площадь, длина окружности
- Cylinder: наследование от Circle, объем
- Конструкторы: по умолчанию, с параметром радиуса

### 10. Класс "Игрок"
- Поля: имя, уровень, опыт
- Конструкторы: по умолчанию, с именем, имя+уровень
- Методы: увеличение опыта и уровня

### 11. Класс "Стек"
- Конструкторы: без параметров, с начальными элементами
- Методы: Push, Pop, Peek

### 12. Абстрактный класс "Фигура" и полиморфизм
- Shape: абстрактный метод CalculateArea
- Rectangle, Circle: переопределение CalculateArea
- Конструкторы для всех классов

### 13. Класс "Часы"
- Поля: часы, минуты, секунды
- Конструкторы: по умолчанию (полночь), часы+минуты, все поля
- Метод инкремента времени

### 14. Классы "Животное" и наследование
- Animal: имя, метод MakeSound
- Dog, Cat, Bird: переопределение MakeSound
- Zoo: список животных

### 15. Класс "Умный дом"
- Поля: температура, влажность, статус сигнализации, освещение
- Конструкторы: без параметров, температура+влажность, все настройки
- Методы управления параметрами
