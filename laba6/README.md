# Лабораторная работа №6

## Тема
Разработка Windows-приложения

## Цель
Научиться разрабатывать Windows Forms приложения, использовать графические интерфейсы для ввода и вывода данных, а также интегрировать и модифицировать ранее созданные классы для создания полноценного приложения.

## Основное задание
Создать Windows Forms приложение как новый проект в решении из лабораторной работы №5.

### Общие требования:
- Запрос подтверждения при выходе из программы
- Подписи компонентов для вводимых и выводимых данных

### Реализуемые формы:

#### 1. Форма "Игра по отгадыванию ответа значения функции"
- Вывод формулы на форму
- Компоненты для ввода/вывода данных с подписями
- Задание количества попыток для угадывания
- Использование классов для:
  - Вычисления значения функции
  - Проверки ввода
  - Расчет времени
- **Таймер** для отслеживания времени на разгадывание (30 секунд)
- **Прогресс-бар** для визуального отсчета времени (опционально)

#### 2. Форма "Работа с одномерными массивами"
- Поле ввода для ручного задания размера массива
- Кнопка для создания массива с длиной по умолчанию
- Кнопка для генерации случайного массива
- **DataGridView** для отображения массива с возможностью ручного редактирования
- Кнопки для операций над массивом:
  - Сортировка
  - Нахождение максимального и минимального значения
  - Среднее арифметическое
  - Выделение цветом максимального и минимального значений в DataGridView
- Модификация и использование класса по работе с одномерными массивами

#### 3. Форма "Игра" (из лабораторной работы №4)
- Использование **DataGridView** или другого компонента для графического отображения игрового поля
- Взаимодействие через клик на ячейку поля (вместо ввода координат)
- Компоненты с подписями для ввода/вывода данных

## Практические задачи (10)

### 1. Класс "Книга" с массивом глав
- **Chapter**: title, content
- **Book**: массив Chapter[]
- **Методы**: 
  - `AddChapter(Chapter chapter)` - добавление главы
  - `GetTotalPages()` - расчет страниц (1 страница = 100 символов)
- Конструкторы с параметрами и без

### 2. Иерархия классов "Фигуры" с массивами
- **Shape** (базовый): `GetArea()`, `GetPerimeter()`
- **Наследники**: 
  - `Circle` (радиус)
  - `Rectangle` (длина, ширина)
  - `Triangle` (стороны a, b, c)
- Массив фигур Shape[], расчет общей площади и периметра

### 3. Класс "Группа студентов"
- **Student**: name, age, grades (массив оценок)
- **StudentGroup**: массив студентов
- **Методы**:
  - `AddStudent(Student student)`
  - `GetTopStudent()` - студент с лучшей средней оценкой
- Конструкторы с параметрами и без

### 4. Иерархия классов "Транспорт" с расчетом времени
- **Transport** (базовый): speed, `GetTravelTime(double distance)`
- **Наследники**:
  - `Car` (расход топлива)
  - `Bike` (ручной привод)
  - `Airplane` (высокая скорость)
- Массив Transport[], расчет времени поездки для заданной дистанции

### 5. Класс "Ученик и оценки"
- **Student**: Name, Grades (массив из 10 оценок)
- **Методы**:
  - `AddGrade(int grade)` - добавление в первую свободную ячейку
  - `GetAverageGrade()` - средняя оценка
- Массив из 5 учеников, вычисление среднего балла

### 6. Класс "Автомобильный парк"
- **Car**: brand, model, mileage, fuelConsumption
- **CarFleet**: массив Car[]
- **Методы**:
  - `AddCar(Car car)`
  - `CalculateTotalFuelConsumption(double distance)` - общий расход топлива
- Конструкторы с параметрами и без

### 7. Класс "Университет и факультеты"
- **Faculty**: name, departments (массив названий кафедр)
- **University**: массив Faculty[]
- **Методы**:
  - `AddFaculty(Faculty faculty)`
  - `GetTotalDepartments()` - общее число кафедр
- Конструкторы с параметрами и без

### 8. Класс "Онлайн-магазин и товары"
- **Product**: name, price, quantity
- **OnlineStore**: массив Product[]
- **Методы**:
  - `AddProduct(Product product)`
  - `GetTotalInventoryValue()` - общая стоимость товаров
- Конструкторы с параметрами и без

### 9. Класс "Банк и клиенты"
- **Client**: name, accountBalance
- **Bank**: массив Client[]
- **Методы**:
  - `AddClient(Client client)`
  - `CalculateTotalBalance()` - общий баланс клиентов
- Конструкторы с параметрами и без

### 10. Класс "Кинотеатр и фильмы"
- **Movie**: title, duration, rating
- **Cinema**: массив Movie[]
- **Методы**:
  - `AddMovie(Movie movie)`
  - `GetLongestMovie()` - самый длинный фильм
- Конструкторы с параметрами и без
